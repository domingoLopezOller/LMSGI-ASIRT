Conector

Un conector eléctrico es un dispositivo para unir circuitos eléctricos.1​ La conexión puede ser temporal, como para equipos portátiles, puede exigir una herramienta para montaje y desmontaje o puede ser una unión permanente entre dos cables o aparatos. Hay cientos de tipos de conectores eléctricos. Están compuestos generalmente de un enchufe y una base.
